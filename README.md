# MetodologIA Website Project

## Project Overview
MetodologIA is a High-Performance Consultancy combining Agile Methods and Artificial Intelligence.
**Motto**: "Success as a Service".
**Core Concept**: "Soberanía Operativa" (Operational Sovereignty).

## Core Structure (v3.1 - Updated 2026-01-28)

### 1. Gateway & Vision
- **index.html**: The Main Gateway. A "Command Center" style entry point directing users to key hubs.
- **vision.html**: The Strategic Manifesto featuring "La Trampa" (Caos + IA = Caos²).

### 2. Business Units
- **Empresas (B2B)**: Corporate Sovereignty, Revenue Operations, AI Sales Teams.
    - Funnel: Diagnóstico -> Workshop (**Paso 1: Try**) -> Bootcamp (**Paso 2: Buy**).
- **Personas (B2C)**: Professional Amplification & Digital Sovereignty.
    - Funnel: Workshop (**Paso 1: Try**) -> Bootcamp (**Paso 2: Buy**) -> Programas.

### 3. Core Hubs
- **Ruta**: The 9-step maturity model with interactive ROI calculator.
- **Campus**: External Learning Management System (campus.metodologia.info).
- **Recursos**: Internal library of Prompts, Templates, and Tools.
- **Servicios**: Global catalog of offerings.

### 4. Brand Identity
- **Legal**: Terms & Privacy policies.
- **Nosotros**: About page, Mission, Ecosystem.
- **Contacto**: Conversion endpoints and booking forms.

## Technical Info
- **Stack**: HTML5, TailwindCSS (local build + CDN fallback), Vanilla JS.
- **Fonts**: Poppins (Headings/Brand), Montserrat (Subtitles).
- **Colors**: Slate-950 (#020617), Brand-Gold (#FFD700).
- **Components**: Web Components (SiteHeader.js, SiteFooter.js).

## SEO/AEO Files
- `robots.txt`: Crawler directives.
- `sitemap.xml`: XML sitemap for search engines.
- `sitemap.html`: Visual interactive sitemap.
- `llms.txt`: AI agent context (Answer Engine Optimization).
- `manifest.json`: PWA configuration.

## Maintenance
- **Standards**: Check `estandares/` for governance and audits.
- **Archive**: Legacy files in `archivado/` directory.
- **Documentation**: `index.txt` files in each directory.
