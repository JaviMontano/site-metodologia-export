# MetodologIA Offering - Service Catalog

## 1. Strategic Purpose
This directory acts as the central menu for the platform. It categorizes the intangible value propositions into concrete, purchasing-ready products. It is critical for the "Consideration" phase of the funnel.

## 2. Content Structure
- **index.html**: The "Offering" Page.
  - **Workshops (Short-term, Tactical)**: "Introducción a IA", "Prompting", "Productividad".
  - **Bootcamps (Mid-term, Strategic)**: "Gestión Comercial", "Marketing con IA".
  - **Consulting (Long-term, Structural)**: "Acompañamiento Estratégico".

## 3. Pricing Strategy
- Displayed prices must be kept in sync with:
  - `bootcamp-ventas-ia.html` (Empresas)
  - Global pricing policies in `legal/terminos.html` (if applicable)

## 4. AEO Optimization
The content is structured to answer: "What services does MetodologIA offer?" and "How much does AI training cost?".
The `Service` schema injection (via `apply_aeo.py`) makes this page highly readable by AI agents.
