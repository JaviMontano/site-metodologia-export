# MetodologIA Services Catalog (Servicios) - Documentation

## 1. Strategic Purpose
Acts as the central directory for all commercial offerings. It provides a structured view of the programs, bridging the gap between Personas (B2C) and Empresas (B2B).

## 2. Content Structure
- index.html: The catalog landing.
  - Features high-level comparisons of different service tiers.
  - Directs users to specific B2B or B2C detail pages.

## 3. Keywords & Topics
- AI Training Programs
- Strategic Consulting
- Empowerment Workshops
- Business Transformation Services

## 4. Maintenance
This file must be kept in sync with the specialized landing pages in /personas/ and /empresas/. Any new product launch must be added here first to ensure global visibility.
