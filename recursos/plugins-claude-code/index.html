¿Qué es un Plugin de Claude Code?

Skills

Instrucciones especializadas que se activan con /comando. Cada skill tiene frontmatter, inputs, pasos y criterios de validación.

Agentes

Subprocesos autónomos con herramientas restringidas y roles especializados. Trabajan en paralelo para análisis profundo, revisión de código o investigación.

Workflows

Pipelines multi-fase que orquestan skills y agentes en secuencia. Discovery completo, análisis de arquitectura o delivery de proyecto — de principio a fin.
