# MetodologIA Evolution Path (Ruta) - Directory

## 1. Strategic Purpose
The "Ruta de (R)Evolución" is the navigational compass of the MetodologIA ecosystem. It allows users to self-diagnose their maturity level and visualize the path from "User" to "Digital Sovereign".

## 2. Maturity Matrix (9 Levels)

### [PHASE 1: METHOD]
1. **Usuario**: Consumer of tools.
2. **Pro**: Efficient operator.
3. **Experto**: Methodological master.

### [PHASE 2: TECHNOLOGY]
4. **Digital**: Tool integrator.
5. **Aumentado**: AI-assisted workflows.
6. **Automatizado**: System builder.

### [PHASE 3: STRATEGY]
7. **Estratega**: Visionary planner.
8. **Líder**: Ecosystem orchestrated.
9. **Soberano**: Autonomous entity.

## 3. Content Structure
- **index.html**: Interactive Roadmap.
  - Visualization: Gantt-style progress bars.
  - Interaction: Hover states reveal detailed "Next Steps".
