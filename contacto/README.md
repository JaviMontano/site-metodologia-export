# MetodologIA Contact Center - Directory

## 1. Strategic Purpose
This is the conversion endpoint for the website. Its primary goal is to reduce friction between "Interest" and "Action", facilitating the booking of discovery calls or direct inquiries.

## 2. Conversion Paths
- **"Primera Conversación"**: 60-minute strategy session (Paid/Free qualification).
- **Direct Support**: WhatsApp and Email channels for immediate resolution.
- **Location**: Physical trust signals (Address, Map).

## 3. Technical Components
- **Booking Widget**: Google Calendar/Calendly embed.
- **Form Validation**: (if applicable).
- **AEO Schema**: `ContactPoint` (Global coverage).

## 4. UX Objectives
- **Load Time**: < 1s (Critical for mobile conversion).
- **Trust**: Visible policy links and professional layout.
