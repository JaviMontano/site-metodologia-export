# MetodologIA Compliance (Legal) - Directory

## 1. Strategic Purpose
This directory ensures regulatory compliance and risk mitigation. It defines the "Rules of Engagement" and protects both the user (Privacy) and the provider (Liability).

## 2. Documents
- **terminos.html**: Terms of Service. Payment policies, IP rights, cancellation rules.
- **privacidad.html**: Privacy Policy. GDPR/CCPA alignment regarding data collection (cookies, leads).

## 3. Criticality
- These documents must be linked from the global footer.
- They are required for ad platform approval (Google Ads, Meta Ads).

## 4. Maintenance
- Review annually or upon significant regulatory changes.
